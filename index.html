FAQ

Частые вопросы

Приложение бесплатное?
+
Да, TimeFlow полностью бесплатен. Никаких платных подписок, скрытых платежей и рекламы нет и не планируется.
Как приложение получает расписание?
+
TimeFlow автоматически парсит официальные сайты университетов и обновляет расписание в реальном времени. Ты просто выбираешь свой вуз и группу — остальное приложение делает само.
Что такое совместный режим?
+
Совместный режим позволяет создавать общие списки задач с одногруппниками. Видно кто над чем работает прямо сейчас, есть push-уведомления об изменениях.
ИИ-ассистент работает офлайн?
+
Нет, ИИ-ассистент требует интернет-соединения — он обращается к Gemini API через облако. Задачи и расписание работают офлайн.
Мой университет не поддерживается. Когда добавите?
+
Напиши нам в Telegram — мы рассматриваем каждый запрос. Сейчас поддерживаются НГУЭУ и НГТУ. Список планируется расширять.
Приложение только для Android?
+
Сейчас да — TimeFlow написан на Kotlin + Jetpack Compose и доступен только для Android. Версия для iOS пока не планируется.
